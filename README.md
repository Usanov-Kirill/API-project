# project
Проект - небольшое API для студентов, в базе данных содержится информация о каждом студенте (student_id, first_name, last_name, grade, phone_number). У каждого студента есть свой id с помощью которого мы можем получить к нему доступ. 

Для просмотра и изменения информации о студенте используются запросы: GET, POST, PUT, PATCH, DELETE.

Также в проекте присутствуют два дополнительных метода (использующих GET запрос):
1. Предоставления случайного студента из общего списка
2. Формирование группы из определённого количества случайных студентов

Ещё добавлены автотесты для эндпоинтов из API.

# Used
Что используется:
1. Используется база данных c инструментом её управления - sqlalchemy
2. Для создания api используется библиотека FastApi, а также requests для запросов к серверу

# Auto-tests
Автотесты используют pytest. 
Во время тестирования вначале создаётся тестовая база данных и сервер, после чего тестируются все запросы к серверу.
В тестах проверяется только возвращенный статус-код запроса.

# Special-metods
Как дополнительные методы были добавленны random_student() и random_group().
1. random_student(). Сначала мы получаем id всех студентов, а дальше при помощи random.choise(), выбирается случайное id из списка и мы возвращаем студета с этим id.
2. random_group(). Принимает на вход кол-во студентов в группе (если подаваемое кол-во больше общего кол-ва студентов то вызываем ошибку), а дальше генерируем случайных студентов как в предыдущем методе пока не наберём нужное кол-во (повторных студентов отбрасываем), в конце возвращаем группу студентов.
